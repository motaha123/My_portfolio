What I've built

Grownerd — AI Educational Platform

A full-stack AI platform for Syrian high school students (grades 9–12 / Baccalaureate). Built solo from scratch — RAG pipeline, Flutter web app, Flask backend, Supabase database, and a custom subscription system.

Live at grownerd.net Flutter Web Flask Supabase RAG Arcjet
Visit live site →
  • RAG system answers questions from official Syrian curriculum textbooks with high accuracy
  • AI quiz mode generates curriculum-aligned questions and tests students interactively
  • Page-flip mode: students navigate any textbook page and get instant AI explanations
  • Supports 8 subjects: Math (1&2), Physics, Chemistry, Biology, Arabic, English, French
  • Subscription system via local Syrian payment methods (Sham Cash, Syriatel Cash, transfer companies)
  • Multi-layer security: Arcjet (bot detection, prompt injection, rate limiting) + Flask-Limiter
  • Guest mode with daily free question limit before requiring signup
  • Auth via Supabase (email/password + Google OAuth) with admin panel for subscription management
